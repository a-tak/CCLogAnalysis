# Server Development Rules

## 責務

- アプリケーションのエントリポイント
- サーバーの起動・設定
- 環境変数の読み込み
- 依存関係の初期化

## 設定管理

- 環境変数から設定を読み取る
  - `PORT`: サーバーポート（デフォルト: 8080）
  - `CLAUDE_PROJECTS_DIR`: Claudeプロジェクトディレクトリ
- デフォルト値を適切に設定
- 起動時に設定内容を表示

## エラーハンドリング

- 起動時の致命的エラーは `log.Fatal` で終了
- 設定の検証を行う
- 適切なログメッセージを出力

## シンプルさを保つ

- ビジネスロジックは書かない
- 依存関係の初期化のみ行う
- ルーティングは `internal/api` に委譲
