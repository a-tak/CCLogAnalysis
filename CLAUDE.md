# Claude Code プロジェクトガイドライン

## パブリック公開プロジェクトの注意事項

このプロジェクトはGitHubでパブリック公開される予定です。以下のルールを遵守してください。

### 個人情報の取り扱い

ドキュメントやコード例で以下の情報を含めないでください：

#### ❌ 禁止事項

- **ユーザー名**: 実際のユーザー名
- **プロジェクト固有の名前**: 実際のプロジェクト名
- **実際のパス**: 個人のディレクトリパス
- **個人を特定できる情報**: メールアドレス、実名など

#### ✅ 代わりに使用するもの

- **ユーザー名** → `{username}` または `user`
- **プロジェクト名** → `my-project` または `{project-name}`
- **パス** → 一般化されたパス例（例: `C:\Users\{username}\...`）
- **ブランチ名** → `feature-branch`, `main`, `develop` などの一般的な名前

### 例

#### ❌ 悪い例

```bash
cd C:\Users\(実際のユーザー名)\.claude\projects\(実際のプロジェクト名)
```

#### ✅ 良い例

```bash
cd C:\Users\{username}\.claude\projects\{project-name}
```

## 開発ガイドライン

### ドキュメント作成時

- サンプルコードやログ例を記載する際は、必ず個人情報を一般化する
- 実際の調査データを使用する場合は、公開前に必ずレビューする

### コミット前チェックリスト

- [ ] ユーザー名が含まれていないか
- [ ] プロジェクト固有の名前が含まれていないか
- [ ] 実際のパスが含まれていないか
- [ ] その他の個人情報が含まれていないか
